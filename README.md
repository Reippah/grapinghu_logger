# 🀄 더 뉴 그라핑후 리그전 기록 생성기

더 뉴 그라핑후 리그전 결과를 입력하면
`더 뉴 그라핑후 기록 및 공지방`에 기록을 바로 작성할 수 있는 텍스트 형식으로 자동 생성해주는 웹 페이지입니다.

아이폰으로는 앱 다운로드를 못해서 제가 쓰려고 만들었어요.

https://reippah.github.io/grapinghu_logger/

---

## 🚀 기능

* 📊 **자동 점수 계산**

  * `(점수 - 25000) / 1000` 방식 적용
* ⏱ **시간 입력**

  * 시작 / 종료 시간 선택 → `0135~0225` 형식 자동 변환
* ✅ **총합 검증**

  * 4인 점수 합이 100000인지 자동 확인
  * 틀리면 결과 생성 불가
* 📱 **모바일 사용 지원**

* 📋 **결과 복사 기능**

---

## 🧾 출력 형식

```text
190
0135~0225
1. 철수 44800 +19.8
2. 영희 33000 +8.0
3. 길동 23900 -1.1
4. 이치히메 -1700 -26.7
```

---

## 🛠 사용 방법

1. 회차 입력
2. 시작 / 종료 시간 선택
3. 1~4등 이름 및 점수 입력
4. 결과 생성 후 복사

---

## ⚠️ 주의사항

* 점수 총합은 반드시 **100000점**
* 기준 점수는 **25000 고정**
* 점수 입력 시 음수 가능 (예: `-1700`)
